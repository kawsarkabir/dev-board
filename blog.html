Question-1: What are the different ways to select an element in the DOM?

The easiest way to access a single element in the DOM is by its unique ID. You can get an element by ID with the getElementById() method of the document object.

Question-2: What is the difference between innerHTML, innerText, and textContent?

In HTML, innerHTML, innerText, and textContent are properties of the DOM (Document Object Model). They allow you to read and update the content of HTML elements.

Question-3: What is event delegation in the DOM?

In the DOM, "event delegation" is a JavaScript design pattern where you attach a single event listener to a parent element instead of adding listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element and then successively triggers on the ancestors (parents) of the target element in the same nesting hierarchy till it reaches the outermost DOM element or document object.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create, add, and remove elements using JavaScript like document.createElement('div'); document.body.appendChild(newElement); newElement.remove();
